## <a name="installation"></a>Instalation:

- Clone the repository or download
- Open whith editor
- Open Terminal
- Install dependencies:
```
npm install
```
- Open database and create:
```
CREATE DATABASE nombreDatabase
```
- Create a file `.env` in the directory and put and complete:
```
NODE_PORT = 3000
DB_PORT = 3306
SERVER = ''
USER = 'root'
PASS = ''
HOST = 'localhost'
DB = ''
SECRET_KEY = ''
REDIS_PORT = 6379
```
- Run the project

`npm run dev`
